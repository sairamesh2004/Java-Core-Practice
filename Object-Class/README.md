# Object Class

This folder contains programs demonstrating concepts related to Java's `Object` class and object cloning.

## Topics Covered

* Object Class Methods
* Object Representation
* Object Comparison
* Object Cloning
* Inherited Methods

## Files

* `ObjectClass.java` – Demonstrates methods inherited from Java's `Object` class.
* `ObjectCloning.java` – Demonstrates object cloning using the `clone()` method.

## Key Concepts Practiced

### Object Class

The `Object` class is the root class of the Java class hierarchy. Every class in Java directly or indirectly inherits from it.

### Common Object Methods

* `toString()` – Returns a string representation of an object.
* `hashCode()` – Returns a hash value for the object.
* `equals()` – Compares objects for equality.
* `getClass()` – Returns runtime class information.

### Object Cloning

Cloning creates a copy of an existing object. Java provides cloning support through the `Cloneable` interface and the `clone()` method.

### Inheritance

Since every Java class inherits from `Object`, understanding its methods is essential for effective object-oriented programming.

## Purpose

These programs were created to explore Java's foundational `Object` class and understand how object cloning and inherited methods are used in real-world Java applications.
