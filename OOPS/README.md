# Object-Oriented Programming (OOP)

This folder contains programs that demonstrate the fundamental principles of Object-Oriented Programming (OOP) in Java.

## Topics Covered

- Classes and Objects
- Constructors
- Constructor Overloading
- Encapsulation
- Inheritance
- Has-A Relationship (Composition)
- Method Overloading
- Method Overriding
- Polymorphism
- Wrapper Classes

## Files

- `Encapsulation.java` – Demonstrates data hiding using private members and getter/setter methods.
- `StudentDetails.java` – Student information management using constructors and objects.
- `multipleConstructors.java` – Constructor overloading examples.
- `ConstructorA.java` – Constructor-related practice program.
- `inheritance.java` – Demonstrates inheritance concepts.
- `ExecutionHas_A.java` – Example of Has-A relationship (Composition).
- `Overloading.java` – Method overloading examples.
- `OverloadingEXP.java` – Practical implementation of method overloading.
- `OverRiding.java` – Method overriding examples.
- `OverRidingEXP.java` – Runtime polymorphism using method overriding.
- `WrapperClass.java` – Demonstrates wrapper classes and primitive type conversions.

## Key Concepts Practiced

### Encapsulation
Bundling data and methods together while restricting direct access to data.

### Inheritance
Acquiring properties and behaviors from an existing class using `extends`.

### Polymorphism
Allowing methods to behave differently based on the object invoking them.

### Composition (Has-A Relationship)
Building complex classes by combining objects of other classes.

### Constructors
Initializing objects during creation and demonstrating constructor overloading.

## Purpose

These programs were created to build a strong understanding of Object-Oriented Programming, which forms the foundation of Java application development, frameworks, and enterprise software.
