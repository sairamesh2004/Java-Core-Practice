# Control Flow

This folder contains programs that demonstrate Java control flow statements and decision-making logic.

## Topics Covered

- If-Else Statements
- Nested If Conditions
- Switch Statements
- Loops (While, For)
- Number Comparisons
- Prime Number Logic
- Leap Year Logic
- Even/Odd Number Checks

## Files

- `Condition.java` – Basic conditional statements.
- `EvenOrOdd.java` – Checks whether a number is even or odd.
- `BigTwo.java` – Finds the largest among two numbers.
- `BigFive.java` – Finds the largest among five numbers.
- `SmallTwo.java` – Finds the smallest among two numbers.
- `SmallFive.java` – Finds the smallest among five numbers.
- `Whileloop.java` – Demonstrates while loop usage.
- `Loop.java` – Loop practice examples.
- `LoopOdd.java` – Prints odd numbers using loops.
- `Switch_Ex.java` – Numeric switch statement example.
- `Switch_Ex_String.java` – String-based switch example.
- `Prime.java` – Prime number checking logic.
- `LeapYear.java` – Leap year validation logic.

## Purpose

These programs were created to practice decision-making, iteration, and control structures in Java. Control flow forms the foundation for implementing algorithms and solving programming problems.
