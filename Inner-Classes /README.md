# Inner Classes

This folder contains programs demonstrating the different types of inner classes available in Java.

## Topics Covered

- Member Inner Classes
- Static Inner Classes
- Method Local Inner Classes
- Anonymous Inner Classes

## Files

- `NestedEXE.java` – Demonstrates member inner classes.
- `MethodLocalEXE.java` – Demonstrates method-local inner classes.
- `StaticInnerEXE.java` – Demonstrates static inner classes.
- `AnonymusInnerClassEXE.java` – Demonstrates anonymous inner classes.

## Key Concepts Practiced

### Member Inner Class
A class defined inside another class that can access members of the outer class.

### Static Inner Class
A nested class declared with the `static` keyword that can be instantiated without creating an object of the outer class.

### Method Local Inner Class
A class declared inside a method and accessible only within that method.

### Anonymous Inner Class
A class without a name that is declared and instantiated in a single statement, often used for implementing interfaces or abstract classes.

## Purpose

These programs were created to understand how nested classes work in Java and how they help organize code, improve encapsulation, and support event-driven programming patterns.
