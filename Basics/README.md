# Java Basics

This folder contains programs that demonstrate fundamental Java concepts and syntax.

## Topics Covered

- Variables and Data Types
- User Input using Scanner
- Type Casting
- Swapping Values
- Basic Output Statements

## Files

- `Variable.java` – Demonstrates Java variables and data types.
- `PrintingName.java` – Basic output example.
- `Swapping.java` – Swapping values between variables.
- `Typecasting.java` – Implicit and explicit type conversion.
- `Scann.java` – Taking user input using Scanner.

## Purpose

These programs were created while learning Java fundamentals and serve as building blocks for more advanced Java concepts.
