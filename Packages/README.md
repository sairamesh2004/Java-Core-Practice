# Packages

This folder contains programs demonstrating the creation and usage of packages in Java.

## Topics Covered

- Package Declaration
- Import Statements
- Accessing Classes from Different Packages
- Package Organization
- Code Modularization

## Files

- `PackageEx1.java` – Basic package creation example.
- `PackageEx2.java` – Package usage and class access example.
- `PackageEx3.java` – Additional package implementation example.
- `OutPack.java` – Demonstrates access to classes outside the current package.
- `OutPack1.java` – Cross-package interaction example.

## Key Concepts Practiced

### Package
A package is a namespace used to organize related classes and interfaces.

### Import Statement
The `import` keyword allows classes from one package to be used in another package.

### Modular Organization
Packages help organize code into logical groups, making large applications easier to maintain and understand.

### Access Control
Packages work together with access modifiers to control the visibility of classes and members.

## Purpose

These programs were created to understand how Java packages help structure applications, improve code organization, and support modular software development.
