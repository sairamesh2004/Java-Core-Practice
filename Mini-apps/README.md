# Mini Applications

This folder contains small console-based applications built while learning Core Java concepts.

Unlike the other folders that focus on individual concepts, these programs combine multiple Java features to simulate simple real-world applications.

## Applications

### Facebook

A basic login validation program that demonstrates user input handling and conditional logic.

### BillPrint

A simple billing system that uses menus, quantities, calculations, and control flow statements.

### Zomato

A restaurant ordering and billing simulation that demonstrates menu-driven programming, nested conditions, and bill generation.

### SMS

A bank transaction SMS generator that simulates account transaction notifications.

## Concepts Practiced

* Scanner Class
* Conditional Statements
* Nested Conditions
* Switch Statements
* Loops
* User Input Handling
* Basic Business Logic
* Console-Based Application Design

## Learning Outcomes

These applications helped in understanding how individual Java concepts work together to build simple interactive programs.

## Purpose

The purpose of these programs is to bridge the gap between concept-based exercises and real-world application development by implementing small, practical use cases using Core Java.
