# Strings

This folder contains programs that explore Java String manipulation and related classes such as StringBuilder and StringBuffer.

## Topics Covered

- String Creation and Manipulation
- String Methods
- Character Operations
- StringBuilder
- StringBuffer
- Mutable vs Immutable Strings

## Files

- `Stringe.java` – Character-wise string processing and case conversion.
- `stringMethods.java` – Demonstrates commonly used methods from the Java String class.
- `stringBufferMethods.java` – Examples of StringBuffer operations such as append, insert, replace, delete, and reverse.
- `stringBuilderMethods.java` – Basic StringBuilder manipulation examples.

## Key Concepts Practiced

- String immutability
- Character extraction using `charAt()`
- String comparison methods
- Searching and substring operations
- Case conversion
- String concatenation
- Mutable string manipulation using StringBuilder and StringBuffer

## Purpose

These programs were created to understand how text processing works in Java and to practice the most commonly used String APIs used in real-world applications.
