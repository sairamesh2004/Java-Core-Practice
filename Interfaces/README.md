# Interfaces

This folder contains programs demonstrating the use of interfaces in Java.

## Topics Covered

- Interface Declaration
- Interface Implementation
- Abstraction
- Polymorphism using Interfaces
- Multiple Behaviors through Interfaces

## Files

- `Interface.java` – Demonstrates interface creation and implementation.

## Key Concepts Practiced

### Interface
An interface is a blueprint of a class that contains abstract methods and constants. It defines what a class should do without specifying how it should do it.

### Abstraction
Interfaces help achieve abstraction by exposing only the required functionality while hiding implementation details.

### Implementation
A class implements an interface using the `implements` keyword and provides implementations for all abstract methods.

### Polymorphism
Interface references can point to objects of implementing classes, enabling runtime polymorphism.

## Purpose

These programs were created to understand how interfaces are used in Java to achieve abstraction, loose coupling, and flexible software design.
