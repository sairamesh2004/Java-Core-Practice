# Exception Handling

This folder contains programs demonstrating exception handling mechanisms in Java.

## Topics Covered

* Try-Catch Blocks
* Multiple Catch Blocks
* Runtime Exceptions
* Error Handling
* Program Stability

## Files

* `exceptionCHECK1.java` – Basic exception handling example.
* `exceptionCHECK2.java` – Demonstrates handling different exception scenarios.
* `exceptionCHECK3.java` – Additional exception handling practice program.

## Key Concepts Practiced

### Exception

An exception is an event that occurs during program execution and disrupts the normal flow of instructions.

### Try-Catch

The `try` block contains code that may generate an exception, while the `catch` block handles the exception and prevents program termination.

### Runtime Exceptions

Exceptions that occur while the program is running, such as division by zero, invalid input, or array index violations.

### Error Handling

Exception handling improves program reliability by allowing applications to recover gracefully from unexpected situations.

## Purpose

These programs were created to understand how Java handles runtime errors and how exception handling can be used to build more robust and reliable applications.
